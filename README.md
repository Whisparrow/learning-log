# Learning Log
Coding learning log — one folder per language, split into notes and solutions.

## Structure
Each language gets its own folder under `notes/`, containing my own practice files and takeaways. Synced Exercism solutions live separately under `solutions/`, organized automatically by language.

## Why this exists
Mainly for me — so I don't lose track of what I've practiced, and so I have a record I can look back on. Not meant to be polished, just a working log of practices.

# Learning Log
Coding learning log — one folder per language, split into lectures and exercises.

## Structure
Each language gets its own folder. Inside, `lectures` holds stuff from lessons/tutorials I'm following, and `exercises` holds practice problems I've worked through.

## Why this exists
Mainly for me — so I don't lose track of what I've practiced, and so I have a record I can look back on. Not meant to be polished, just a working log of practices.
